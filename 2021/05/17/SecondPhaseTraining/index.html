二階段軍事訓練 [2021最新/4個月義務役]

2021-05-17
留學資訊 / 兵役
Word count: 1.1k   |   Reading time≈ 3 min

來看看後三個月都在幹嘛吧!


歷經一個月終於來到了第二階段的專長訓! (也就是之前的下部隊)

專長訓顧名思義就是你會學一些特殊專長，

像我自己就是66火箭彈兵&T85榴彈兵，這兩個專長。

當然不同的單位有不同的訓練，

底下將以66火箭彈兵&T85榴彈兵，這兩個專長訓作介紹，

就讓我們繼續看下去後面三個月的訓練菜單吧!


專長訓目錄

  • 打靶 [175公尺臥姿射擊]
  • 專長兵器教練
  • 手榴彈投擲 [窗投]
  • 500障礙
  • 格鬥術
  • 戰技體能 [傷患搬運]
  • CQB城鎮戰
  • 行軍


# 打靶 [175公尺臥姿射擊]

訓練內容

​ - 和新訓時的訓練大同小異，請參閱上一篇 一階段基本訓 。

如何不被電爆

​ - 到這個階段大家打靶應該滿熟練了，所以好好做就對了。

​

後面熟習打靶後，大家就會開始比較子彈上靶數，

然後你就會看到一堆神人，明明一人打9發，靶紙上卻有18發彈孔XDD。

這些人不加入實戰單位真的太可惜了。


# 專長兵器教練

訓練內容

​ - 訓練如何拆解、組裝、使用T85榴彈發射器。
​ - 實際操作 66火箭彈 & T85榴彈發射器 的使用方式。

如何不被電爆

​ - 這個課程只有一開始覺得新鮮，等你每天都要上一樣的課程時，就自己找樂子吧。

​

還有不要怪長官啦，這種操作一次就會的東西，但他們得按表操課所以也是很無奈。


# 手榴彈投擲 [窗投]

訓練內容

​ - 和基本訓的手榴彈投擲不同 (若不清楚請參照 “基本訓” 的內容)，此訓練為高跪姿將手榴彈投入窗口。

如何不被電爆

​ - 我個人覺得蠻難投進，因為你要力氣大之外，還要能精準地投入窗口，而且跪姿還蠻難施力的。


# 500障礙

訓練內容

​ - 在時間內完成各項關卡 (爬竿、爬牆、爬木頭高牆、爬過鐵絲網…)

如何不被電爆

​ - 比起前面的內容算蠻好玩的訓練，當然也會比較累。

​

有懼高症的朋友，請有心理準備，我就有遇到在木頭高台上待了20分鐘還不敢跳下來，

甚至連班長都上去開導他自己跳了三次，還是無法說服他跳。


# 格鬥術

訓練內容

​ - 基本防身術和格鬥體技，訓練單兵徒手攻擊和防衛能力。

如何不被電爆

​ - 小心別把鄰兵的手折了哈哈。


# 戰技體能 [傷患搬運]

訓練內容

​ - 訓練訓員在戰場上如何搶救傷患，包含單人搬運、雙人搬運…。

如何不被電爆

​ - 沒錯，這是體能課程，搬運技巧就那些，主要就是要鍛鍊你的身體素質來救人。


# CQB城鎮戰

訓練內容

​ - CQB (Close Quarter Battle)，又稱近距離作戰，此訓練主要讓訓員對於城鎮作戰有更多認識。

如何不被電爆

​ - 現在很多戰場是在城鎮中，而台灣地又小城鎮密集，所以我覺得是個算蠻有用的課程訓練。


# 行軍

訓練內容

​ - 全副武裝挾槍行走。(不同單位也許會有夜行軍、急行軍)

如何不被電爆

​ - 挾槍行軍手真的酸，要偷吃步 (彈夾塞在S腰帶) 就不要被抓到，不然有可能叫你拿兩把槍走哈哈哈。

​ - 誠心建議不要當交管，很累要一直跑，也沒什麼福利好拿，就算有通常也要當到結訓，CP值超低。




這些差不多就是你在專長訓會遇到的課程了，

相比一階段，我覺得二階段的課程有趣多了，也比較實用。

後面三個月通常會比前面一階段更輕鬆啦，起碼比較不無聊，也通常比較自由，


講完訓練內容，來看看軍中訓練以外的生活吧!
